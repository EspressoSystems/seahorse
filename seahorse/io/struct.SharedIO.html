Struct seahorse::io::SharedIO[−][src]

pub struct SharedIO { /* fields omitted */ }
Expand description

Wrapper around an input stream and an output stream

Both the input and output are trait objects, so any types implementing Read and Write can be used here.

SharedIO also has an internal buffer so that it can implement BufRead.

Create a SharedIO instance using a bidirectional pipe.
