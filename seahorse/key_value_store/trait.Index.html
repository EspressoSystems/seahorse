Expand description

An Interface for an in memory index which can insert and remove items To create a persistable index type you can implement this trait for whatever in memory index you choose to use. E.g. if you create a new type called CustomHashMap<K,V> you can implement Index<(K,V)> for it and then use the persitable type PersistableMap<(K,V), CustomHashMap<K,V>>.
